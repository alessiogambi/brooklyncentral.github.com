CassandraDatacenter A group of CassandraNodes -- based on Brooklyn's DynamicCluster (though it is a "Datacenter" in Cassandra terms, where Cassandra's "cluster" corresponds to a Brooklyn Fabric, cf CassandraFabric).
CassandraFabric A fabric of CassandraNodes, which forms a cluster spanning multiple locations.
CassandraNode An Entity that represents a Cassandra node in a CassandraDatacenter.
